# Générateur de certificat de déplacement

## Développer

### Installer le projet

```console
$ git clone https://github.com/cristianpb/covid-19-certificate.git
$ cd covid-19-certificate
$ npm i
$ npm start
```

### Générer le code de production

```console
$ npm run build
```

Le code à déployer sera dans le dossier `dist`
